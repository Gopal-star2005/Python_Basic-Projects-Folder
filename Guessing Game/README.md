# 📘 Overview
The **Number Guessing Game** is a Python-based command-line game where the computer randomly selects a number between **1 and 100**, and the player must guess the correct number within **10 attempts**.

The program provides feedback after each guess to guide the player toward the correct answer.

# 🚀 Workflow

1. Program generates a random number between **1 and 100**
2. Player enters a guess
3. Program checks the guess
4. Player receives hints:
   - Too high
   - Too low
5. Player continues guessing until:
   - Correct number is guessed
   - All attempts are used


# 🎮 Game Rules

1. Guess a number between 1 and 100
2. You have 10 attempts
3. Hints will help guide your guesses
4. The game ends when you guess correctly or run out of attempts
