# 🚀 Projects Included

## 🧮 1. Simple Calculator

A command-line calculator that performs basic arithmetic operations.

### Features

* Addition
* Subtraction
* Multiplication
* Division
* Division-by-zero error handling
* Continuous calculations via CLI

### Concepts Demonstrated

* Python functions
* Conditional statements
* User input handling
* Exception handling

---

## 🎯 2. Number Guessing Game

An interactive command-line game where users attempt to guess a randomly generated number.

### Features

* Random number generation
* Guessing range between **1 and 100**
* Maximum **10 attempts**
* Hint system (Too High / Too Low)
* Replay option

### Concepts Demonstrated

* Random module
* Game logic
* Loop control
* Input validation

---

## 📄 3. Word Counter

A Python utility for analyzing text files and generating word statistics.

### Features

* Count total words
* Count characters
* Count characters excluding spaces
* Count lines
* Preview first 20 words
* Sample text file generation
* File error handling

### Concepts Demonstrated

* File handling
* Text processing
* Exception handling
* CLI utilities

# ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Gopal-star2005/Python-Projects-Folder.git
cd Python-Projects-Folder
```

Navigate to any project folder and run the Python script.

Example:

```bash
cd Calculator
python calculator.py
```
