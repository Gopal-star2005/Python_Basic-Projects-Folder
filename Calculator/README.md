# 📘 Overview
The **Python Simple Calculator** is a command-line application designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

The project demonstrates **core Python programming concepts** including:

- Functions
- User input handling
- Conditional statements
- Exception handling
- Loop-based interaction

It serves as a **foundational programming exercise** for understanding structured Python application design.

---

# 👤 About the Project
This project was created as part of a **Python programming practice task** to demonstrate:

- Modular function design
- Clean command-line interface interaction
- Basic error handling techniques
- Control flow using loops and conditionals

The application allows users to repeatedly perform calculations until they choose to exit.

---

# ✨ Key Features
- ➕ Addition operation
- ➖ Subtraction operation
- ✖️ Multiplication operation
- ➗ Division operation
- ⚠️ Division-by-zero error handling
- 🔁 Continuous operation until user exits
- 🖥️ Simple and interactive CLI interface

---

# 🛠 Tech Stack
- **Python** — Core programming language
- **CLI (Command Line Interface)** — User interaction
- **Exception Handling** — Error management

---

# 🚀 Workflow
1. User selects an arithmetic operation
2. Program prompts for two numbers
3. Operation is executed using modular functions
4. Result is displayed to the user
5. Program repeats until user chooses to exit
